## What It Is

## Standout Features (Why Recruiters & Clients Will Lose Their Minds)

- **Stunning NEXUS Aesthetic**: Neon green `#00ff87`, cyan `#60efff`, hot pink `#ff0080`, yellow `#ffcc00` on deep cyberpunk dark. Glassmorphism, glowing effects, buttery animations.
- **Interactive Neural Brain Visualizer** (Canvas): 10 agent nodes + core with real-time data flow particles, organic idle movement, click-to-inspect, pulse animations when analyzing. This alone is a conversation piece.
- **Real Canvas Computer Vision**: Drag-and-drop or click-to-upload photos in 10 required categories. Uses HTML5 Canvas + ImageData to compute brightness, saturation, edge density, and domain-specific "grease index" + auto-flags (dark, blurry, residual grease, etc.). No external AI needed — pure client-side magic.
- **Live What-If Simulator**: Sliders and selects for grease remaining, fan hinge condition, access panels, rush factor. Scores, risks, and brain visualization update instantly. Demonstrates deep algorithmic understanding.
- **Full 10-Agent Parallel Review**: Every agent (QA Inspector, Safety, NFPA, Revenue, Training, Fraud, Customer Advocate, Ops, Regional, Executive) produces independent scores, findings, and recommendations. Click any agent card or node for deep dive modal.
- **Pattern Recognition & Fraud Detection**: Timeline Canvas visualization with clustering detection. Historical pattern matching via localStorage. Auto-flags rushed jobs, photo padding, missing critical categories.
- **Dynamic Scoring Engine**: Implements every rule from the Hood Boss QA protocol — special escalation flags, root cause logic, risk matrix, future 30-day prediction probabilities.
- **Executive Report Generator**: One-click beautiful formatted report matching the exact management template. Print-to-PDF ready with clean styles. Also exports full JSON package.
- **Coaching Engine**: Auto-generates targeted, actionable coaching items and training recommendations.
- **PWA-ready**: Add to Home Screen on mobile/desktop. Custom SVG icon that looks incredible. Offline capable after first load (Tailwind + Chart.js via CDN).
- **Keyboard Shortcuts**: `/` focuses location, `⌘+Enter` activates brain, `?` opens full report.

## How to Use (30 seconds to wow)

1. Open `index.html` in any modern browser (Chrome, Safari, Firefox, Edge).
2. Click **LOAD SAMPLE** — instantly populates a realistic flawed job (rushed, missing fan photos, residual grease, missing access panels, timeline anomalies).
3. Watch the **NEXUS Brain Canvas** come alive with data pulses.
4. Click any agent node or card to see its specialized analysis.
5. Tweak the **What-If sliders** — scores and visualization react live.
6. Hit **ACTIVATE NEXUS BRAIN** (or ⌘+Enter) for full 10-agent synthesis.
7. Click **VIEW FULL EXECUTIVE REPORT** → **PRINT / PDF** for a management-ready document.
8. **DOWNLOAD REPORT PACKAGE** gives you the complete JSON + opens the printable version.
